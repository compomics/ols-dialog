About the OLS Dialog

OLS Dialog 3.0 is a Java front end to the Ontology Lookup Service ( OLS 3.0 ), originally developed by Harald Barsnes, University of Bergen, Norway.

and extended for the new OLS 3.0 by Yasset Perez-Riverol, Proteomics, PRIDE, EMBL-EBI, UK. Project web page: https://github.com/PRIDE-Toolsuite/ols-dialog.

More information about the OLS can be found here.

Click the help icon in the bottom left corner for help.

Any further questions can be forwarded to the PRIDE team at the EBI: pride-support@ebi.ac.uk.
